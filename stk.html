Statens Televisionskanal

Individuell inlämningsuppgift där målet var att, med hjälp av JavaScript, HTML och CSS (i mitt fall SCSS/Sass), ta fram en hemsida för en fiktiv tv-kanal där man skulle kunna lägga till samt söka på tillagda tv-program. Uppgiften ställde även krav på att reflektera kring/implementera tillgänglighetsaspekter enligt WCAG. Programmen lagras i local storage.

IDE: Visual Studio Code
